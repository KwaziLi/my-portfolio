# Отчёт: Развёртывание сайта Hugo на GitHub Pages

## Что было сделано

1. Установлена CMS Hugo на локальный компьютер (macOS, hugo-extended)
2. Создан новый сайт Hugo командой `hugo new site my-portfolio`
3. Подключена тема Ananke как Git-подмодуль
4. Создан контент сайта: главная страница, «О себе», «Проекты»
5. Настроен CI/CD через GitHub Actions для автоматической сборки и деплоя
6. Сайт развёрнут на GitHub Pages

## Технологии

| Технология | Назначение |
|---|---|
| Hugo | Генератор статических сайтов |
| Ananke | Тема оформления |
| GitHub Actions | CI/CD |
| GitHub Pages | Хостинг |
